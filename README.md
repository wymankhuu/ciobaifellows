# CIOB AI Design Fellows

A showcase platform empowering educators to build agency and design AI tools for higher-order thinking within their unique school contexts.

## Overview

This application serves as a digital portfolio and showcase for the CIOB AI Design Fellows. It highlights the innovative AI applications developed by educators from Consortium, International, and Outward Bound schools. Each profile provides deep insight into the fellow's journey, the specific AI tool they developed, and the pedagogical rationale behind it.

## Features

- **Fellow Profiles**: Interactive cards showcasing each fellow, their role, and school.
- **Project Showcases**: Detailed views of each AI tool, including video introductions, app descriptions, and impact statements.
- **Responsive Design**: Fully optimized interface that adapts seamlessly from desktop to mobile devices.
- **Category Filtering**: Organization of fellows by school network (Consortium, International, Outward Bound).

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI, Lucide React
- **Animation**: Framer Motion
- **Routing**: Wouter

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd ciob-ai-fellows
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5000` to view the application.

## Project Structure

- `/client`: Frontend source code
  - `/src/components`: Reusable UI components
  - `/src/pages`: Main application pages
  - `/src/lib`: Utility functions and data definitions
- `/shared`: Shared type definitions
